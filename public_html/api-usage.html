Chicha Isotope Map — API Guide

Detailed documentation for the JSON endpoints served from __API_ROOT__ on __DISPLAY_HOST__.

Base URL

All requests begin with __API_ROOT__. When the server runs with a configured domain the same endpoint is available over HTTPS.

The examples below assume __BASE_URL__; replace it with your preferred host or short-link.

Working with /api/latest

The endpoint accepts decimal degrees for lat and lon, a search radius in metres (radius_m), and an optional limit (default 25, max 200). Results are sorted from newest to oldest.

curl -s "__BASE_URL__/api/latest?lat=35.6804&lon=139.7690&radius_m=1500&limit=10" | jq '.markers[0]'
curl -s "__BASE_URL__/api/latest?lat=51.3890&lon=30.0990&radius_m=2200&limit=15" | jq '.returned'

Track catalog

Use /api/tracks for alphabetic listings, /api/tracks/years/{year} or /api/tracks/months/{year}/{month} for time windows, and /api/tracks/index/{n} for deterministic downloads.

curl -s "__BASE_URL__/api/tracks?limit=25" | jq '.tracks[0].apiURL'
curl -s "__BASE_URL__/api/tracks/years/2024" | jq '.tracks | length'

Track payloads

Each track streams as newline-delimited JSON with the .json suffix so browsers save the file. Optional from and to parameters filter the marker ID range. Older .cim URLs keep working for compatibility.

curl -L -o sample.json "__BASE_URL__/api/track/ABC123.json"

JSON archive

When -json-archive-path is set, the node builds a tgz bundle on a schedule (daily, weekly, monthly, or yearly). Download it at __BASE_URL____ARCHIVE_ROUTE__.

curl -L -o archive.tgz "__BASE_URL____ARCHIVE_ROUTE__"

Short links

The /api/shorten endpoint issues and finalises shareable URLs for map views. Submit JSON with url, optional code, and commit.

curl -s -X POST "__BASE_URL__/api/shorten" \
  -H 'content-type: application/json' \
  -d '{"url":"/", "commit":false}' | jq

Rate limiting & conventions

  • Responses may include X-RateLimit-Notice and X-RateLimit-Delay headers when the per-IP queue slows requests for fairness.
  • Timestamps are Unix seconds. Some sources ship millisecond precision and are normalised automatically.
  • Dose rates are in µSv/h; count rates are in CPS. Every payload includes a disclaimers map to surface the licence notice in your app.
